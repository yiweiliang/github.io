Weather App
Check weather anytime, anywhere
From An Idea to Implementation

The design process started by asking a question: How can I improve a communication process?


For this particular project my plan was to create a interactive weather app that could help people realize the weather conditions of the place ther are in. CSS animations were used to creatve an interactive interface. I have used PHP to parse weather data using the APIs of Open Weather, the site is still under development and in the future I will try to add interactive weather backgrounds using Flickr's photo APIs so that they will change according to weather information.
